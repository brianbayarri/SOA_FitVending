ENTRADAS

1. Fotoresistor: Se encarga de darnos informacion sobre la iluminacion del ambiente donde se encuentra la vending y determinar si es necesario o no encender una luz interna para que el usuario pueda visualizar con mayor facilidad el producto a consumir
2. Sensor de movimiento: Determinara si el producto fue retirado o no de la maquina vending
3. Sensor infrarrojo: Utilizado para informar si hay stock o no de un producto
4. Sensor de temperatura LM35: Encargado de informarnos la temperatura interna de la vending

SALIDAS

1. Servomotor sg 90: Utilizado para giro de resortes a la hora de expender un producto seleccionado
2. Luz led roja: Indicara que un producto aun no puede retirarse
3. Luz led verde: Indicara que un producto esta listo para retirarse
4. Display LCD 16*2 con módulo l2C: Informara al usuario la temperatura de los productos como asi tambien cuando un producto esta listo para retirarse
5. Tira de led RGB: Encargada de iluminar la vending cuando se encuentra en un ambiente oscuro
