# Version 1.0.1

## Release date: 01/07/2019
 	
### Descripción:
Aplicacion desarrollada para el control de alimentacion y ejercicio de una persona para que la misma pueda pedir un producto de la maquina Vending.

### Bugs:
- Sin bugs reportados.

### Added:
- Pantalla de alimentacion como fragment
- Pantalla de cronometro como fragment
- Pantalla de logros como fragment
- Pantalla de perfil como fragment
- Pantalla de rutina como fragment
- Pantalla principal como fragment
- Listener para activar el SHAKE
- Listener para el sensor de proximidad
- Contador de pasos
- Conexion a la base de datos
- ABM del usuario
- Control de calorias
- Comunicacion con Arduino
- Validacion de stock
- Pantalla de carga
- Conexion bluetooth

### Removed:
